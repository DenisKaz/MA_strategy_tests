### Moving Average (MA)

–ü—Ä–æ—Å—Ç–∞—è —Å–∫–æ–ª—å–∑—è—â–∞—è —Å—Ä–µ–¥–Ω—è—è (SMA) –∑–∞ N —Å–≤–µ—á–µ–π. –ò—Å–ø–æ–ª—å–∑—É–µ—Ç—Å—è –¥–ª—è –≤—ã—è–≤–ª–µ–Ω–∏—è —Ç—Ä–µ–Ω–¥–∞ –∏ —Å–∏–≥–Ω–∞–ª–æ–≤ –ø–µ—Ä–µ—Å–µ—á–µ–Ω–∏—è.

- **–ë—ã—á–∏–π** (1): –±—ã—Å—Ç—Ä–∞—è MA > –º–µ–¥–ª–µ–Ω–Ω–∞—è MA
